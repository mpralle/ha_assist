# HA Assist

A custom assistant add-on for Home Assistant.

## Configuration

| Option      | Description                          | Default |
| ----------- | ------------------------------------ | ------- |
| `log_level` | Logging verbosity (trace → fatal)    | `info`  |

## How it works

This add-on connects to the Home Assistant API and provides assistant capabilities.
